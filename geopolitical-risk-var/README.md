## Overview

This project analyzes how geopolitical shocks affect the daily volatility and Value at Risk (VaR)
of a commodity portfolio listed on B3, covering major events from 2022 to 2026.

## Assets

| Ticker | Company | Commodity |
|--------|---------|-----------|
| VALE3 | Vale S.A. | Iron ore / Nickel |
| PETR4 | Petrobras | Oil / Gas |
| PRIO3 | PetroRio | Offshore oil (E&P) |
| GGBR4 | Gerdau | Steel / Scrap |

## Methods

- Historical and Parametric VaR (95%)
- Rolling volatility (63-day window)
- Stress-period correlation matrices
- Geopolitical event timeline analysis

## Tools

Python, yfinance, NumPy, pandas, matplotlib
